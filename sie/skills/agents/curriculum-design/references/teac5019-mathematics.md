## PEDAGOGICAL FRAMEWORK — KEY CONCEPTS

### Constructivist Theory (Tutorial 3)
- Knowledge is actively constructed by the learner, not passively received
- Students develop autonomous, self-motivated mathematical activity
- Teacher as facilitator — guides, doesn't transmit
- Mathematical truths are cooperatively established within a culture
- Skemp's distinction: instrumental understanding (rules/procedures) vs relational understanding (why rules work) — target is relational

### The "Fading Technique" / Three-Times Rule (Tutorial 3)
Each concept type presented THREE times:
1. Teacher presents worked example with discussion (NOT copied into books)
2. Same example with different numbers, students provide input (NOT copied)
3. Students copy the question, complete in books, most reluctant students asked for input
Then: one identical question → homework

### Homework Principle
Homework consolidates learning — never introduce new or difficult content. Aligns with ZPD (Vygotsky).

### Teaching and Learning Cycle (Tutorial 4)
For unit planning: diagnostic → planning → teaching → assessment → reflection
Pivotal in Teacher Performance Assessment (TPA)

### Differentiation Strategies (Tutorial 4)
- Curriculum compacting
- Providing key vocabulary
- Developing individual learning goals
- Learning centres (guided or independent)
- Tiered and levelled activities
- Interest centres
- Learning contracts
- Problem-solving and challenge-based learning
- Open-ended questioning
- Group and independent study

### Assessment Types (Tutorial 4)
- **Diagnostic** — before teaching; identifies prior knowledge and learning difficulties (quiz, worksheet, discussion, Five Question Approach)
- **Formative (most effective)** — during teaching; directly influences instruction; Five Question Approach designed to be formative
- **Summative** — end of unit; evaluates learning; "Summative becomes formative" — first error occurrence guides next teaching cycle

### Feedback Research
- Deci et al (1999): Positive feedback increases likelihood students return to/persist in activity and self-report higher interest
- Hattie (2007): Feedback combined with effective instruction is very powerful in enhancing learning
- Type of assessment can reduce student confidence — avoid "one mark per step" marking scales

### Five Question Approach (Tutorial 3)
Structured teacher questioning strategy — designed to be formative based on student understanding and performance.

### Student Confidence Reducers (Tutorial 4)
- Setting difficult tests with unfair marking scales
- Doing easy questions in class, hard ones for homework
- Presenting examples in complicated manner
- Too many same-type questions at same time
- Never revisiting areas where students had difficulty

### Literacy in Mathematics (Week 3 Lecture)
Four modes: reading/viewing, writing, speaking, creating print/digital materials
Includes: mathematical vocabulary, conventions for communicating in symbols, interpreting tables/graphs

### Numeracy in Mathematics (Week 3 Lecture)
- Embedded throughout Mathematics K–10 Syllabus (not tagged separately)
- Involves selecting and critically evaluating mathematics use

### ICT in Mathematics (Week 3 Lecture + Tutorial 3)
- Tools: calculators, spreadsheets, dynamic geometry (GeoGebra), computer algebra and graphing software
- Students develop ICT capability: information access/management, creation/presentation, problem-solving, decision-making, creative expression, empirical reasoning
- Example activity: Mobile phone plan comparison (Juan vs Sylvia) — real-world context for linear functions using spreadsheet

### General Capabilities (Week 3 Lecture)
Critical and creative thinking, Ethical understanding, ICT, Intercultural, Literacy, Numeracy, Personal and social

## PROVEN 5-WEEK UNIT DESIGN: Algebraic Techniques → Functions

### Design Rationale
Algebraic techniques is a foundational Stage 5 thread → serves as prerequisite for Functions path. Natural bridge: substitution (alg) → function notation f(x) → graphing → domain/range.

### Week Structure
| Week | Core Focus | Path Introduction | Formative |
|------|-----------|------------------|-----------|
| 1 | Algebraic fractions | — | Exit ticket |
| 2 | Expanding | — | Think-pair-share |
| 3 | Factorisation | f(x) notation preview | Mini quiz |
| 4 | Algebraic fractions practice | GeoGebra graphing | Problem-solving task |
| 5 | Review + integration | Mobile plan investigation | Formative test |

### Differentiation Three-Tier Model
- **Tier 1 (all):** Core content with fading technique, structured worksheets, exit tickets
- **Tier 2 (ready):** Extension prompts — qualitatively different tasks, not "more questions"
- **Tier 3 (path):** Challenge envelope within lessons, small-group conferencing, GeoGebra extension

## THREE RESOURCES — DESIGN PATTERN

### Resource 1: Formative Assessment
- Exit ticket: 5 items, 5 minutes, end of each lesson
- Covers: simplifying fractions, expanding, factorising, substitution, word problem
- Google Forms (auto-scoring + dashboard) + PDF backup
- Color-coded by difficulty (green/amber/red)
- Final item = "explain your thinking" (Working Mathematically)
- Timely feedback in next lesson (Hattie 2007)

### Resource 2: Digital Artefact (ICT)
- GeoGebra Classic activity: graphing linear functions
- Sliders to explore m and c in y = mx + c
- Mobile plan comparison (graph two plans, find break-even)
- ICT skills: sliders, axis scaling, labelling points, screenshot/export
- Scaffolding for EAL/D: key vocabulary highlighted with visual examples

### Resource 3: Path Resource
- Mobile phone plan investigation (extended from TEAC 5019 Tutorial 3)
- Original: spreadsheet in tutorial → GeoGebra + piecewise functions in this version
- Task: write plans as functions, graph on GeoGebra, find break-even algebraically + graphically, interpret, extension to piecewise functions
- Integrates algebra (solve equation) + ICT (GeoGebra) + literacy (written recommendation)
- Success criteria card + worked example included
