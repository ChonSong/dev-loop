# Solving the Tandem Clock‑Grid CAPTCHA

This skill describes how to solve the 6‑image clock‑grid CAPTCHA that appears in the Tandem viewer when applying to Amazon.jobs roles.

## Steps

1. **Locate the grid** – In the Tandem viewer, find the 6‑image clock grid (circular arrangement like a clock face). Each image shows a number from 1 to 9.
2. **Read the numbers** – Identify the number shown in each image. Note the sequence.
3. **Click clockwise** – Starting at the 12 o’clock position, click the numbers in clockwise order (e.g., 12 → 3 → 6 → 9). Follow the exact sequence; do not guess.
4. **Confirm** – After the last number is clicked, click the “Confirm” button.
5. **Proceed** – The application form will load; fill in the required details.

## Common Pitfalls

- **Wrong order** – Clicking numbers out of the displayed clockwise sequence will cause failure.
- **Premature clicks** – Wait for the new image to appear before clicking the next number.
- **Disabled Confirm** – If the “Confirm” button stays disabled, verify that all six numbers have been clicked in the correct order.

## Verification

After clicking “Confirm”, the application form should appear with fields for personal information. If it does not, repeat the CAPTCHA steps.
