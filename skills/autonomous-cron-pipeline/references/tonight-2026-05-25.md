### "Invalid tunnel secret" errors — cosmetic, not real failures
- **Finding:** `journalctl --user -u hermes-webui-tunnel -f` shows recurring `ERR failed to serve incoming request error="Unauthorized: Invalid tunnel secret"` and `ERR Register tunnel error from server side error="Unauthorized: Invalid tunnel secret"` every ~30s.
- **Root cause:** These are random public IP addresses hitting Cloudflare's edge — scanners/bots that stumble onto the tunnel URL. Cloudflare rejects them with the generic "invalid secret" message. The tunnel itself is working fine.
- **Evidence:** `curl http://localhost:8787/` returns `<!doctype html>` — tunnel IS serving traffic locally.
- **Verdict:** No action needed. This is background noise, not a service failure.
